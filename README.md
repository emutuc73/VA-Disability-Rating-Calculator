# VA Disability Rating Calculator

A simple web-based calculator that estimates a veteran's combined VA disability rating using the VA's "whole person theory."

This tool is designed to help veterans quickly estimate their combined disability rating before submitting or reviewing a VA claim.

## Live Calculator

Use the calculator here:

## Features

- Calculates combined VA disability ratings
- Supports multiple disability percentages
- Applies VA whole-person math logic
- Instant calculation in the browser
- Mobile-friendly design

## How It Works

The Department of Veterans Affairs does not simply add disability percentages together.

Instead, the VA applies the **Whole Person Theory**, where each additional disability is calculated from the remaining non-disabled portion of the veteran.

Example:

- 50% disability leaves 50% efficiency remaining
- A second 30% disability applies to that remaining 50%
- Result: 65% combined rating (rounded to nearest 10%)

This calculator automates that process.
